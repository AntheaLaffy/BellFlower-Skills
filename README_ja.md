# 如月風鈴のskills
[en](README_en.md) | [zh](README.md) | [ja](README_ja.md)
## 紹介
これは私がClaude Codeで遊びながら書いたいくつかのスキルドキュメントで、大学生活で直面したさまざまな問題を解決するためのものです。

私のPCはArch Linuxを使っています。あなたの環境とは異なるかもしれませんが、もし役に立てば嬉しいです。

## インストール
特別な手順はありません。リポジトリ全体をクローンして、Claude Codeの`skills`フォルダに入れてください。

もちろん、プロジェクトの`skills`フォルダに入れても構いません。

---

## ドキュメント処理関連
- `pdf-math-convert`：PDFをMarkdownテキストに変換し、さらに編集しやすくします。
> 特徴：1. 数式をLaTeX形式に変換することをサポート  2. 画像をBase64で埋め込む
- `markdown-translate`：LLMでMarkdownテキストを翻訳し、LaTeX数式・コードブロック・画像・書式を保持します
> 特徴：1. 数式/コード/画像を保護して翻訳  2. 複数ファイルの並列翻訳対応  3. 翻訳後の整合性チェック付き
